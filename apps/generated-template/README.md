# generated-template

Starter copied by `@viberun/companion` when it initializes a new project
workspace. Do not edit files here to customize a user's project — the
Companion's `implement_slice` handlers write to the user's own workspace under
`~/.viberun/projects/<id>`.

Placeholder tokens the Companion replaces on copy:

- `{{APP_NAME}}` — display name
- `{{APP_SLUG}}` — URL/package slug
- `{{APP_PITCH}}` — one-sentence description
